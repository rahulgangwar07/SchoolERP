Available Titles For EXCEL / CSV Sheet

Excel Column Fields Required for Upload
  • 1. book_type – Type of book (e.g., Textbook, Reference)
  • 2. title – Title of the book
  • 3. isbn_no – ISBN number (optional)
  • 4. author – Author name
  • 5. edition – Edition info (e.g., 1st, Revised)
  • 6. volume – Volume number (if any)
  • 7. publisher – Publisher name
  • 8. price – Price of the book (e.g., 299.99)
  • 9. no_of_copies – Total number of copies
  • 10. no_of_pages – Total pages in the book
  • 11. almeria_no – Almirah number for storage
  • 12. rack_no – Rack number
  • 13. position – Position on the rack (optional)
  • 14. accession_no – Unique accession number (e.g., ACC12345)
  • 15. book_language – Language of the book (e.g., English, Hindi)
  • 16. class_id – ID of the class (numeric)
  • 17. subject_id – ID of the subject (numeric)
